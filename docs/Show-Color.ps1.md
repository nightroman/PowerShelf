# Show-Color.ps1

```
Shows all color combinations, color names and codes.
Author: Roman Kuzmin
```

## Syntax

```
Show-Color.ps1
```

## Description

```
The script shows all combinations of background and foreground colors as a
table. The last columns are hexadecimal and decimal codes and color names.
```

## Inputs

```
None.
```

## Outputs

```
None.
```
