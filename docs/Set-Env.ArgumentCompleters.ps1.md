# Set-Env.ArgumentCompleters.ps1

```
Completes Set-Env.ps1 -Name, -Value .
Author: Roman Kuzmin
```

## Syntax

```
Set-Env.ArgumentCompleters.ps1
```

## Description

```
`Name` is completed with environment variable names.
`Value` is completed with the current location item paths.
```
